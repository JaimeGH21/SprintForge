# SprintForge

Repositorio del proyecto **SprintForge** para la asignatura de Ingeniería del Software II.

## Descripción

Este repositorio se ha creado para gestionar el desarrollo del proyecto de la asignatura.  
Aquí iremos subiendo el código, la documentación y los diferentes cambios que hagamos durante las prácticas.

La idea es utilizar Git y GitHub para trabajar en equipo y llevar un control de versiones del proyecto.

## Integrantes

- Jaime García
- Erika Macarro

## Objetivo

El objetivo de este proyecto es aprender a trabajar con repositorios, control de versiones y desarrollo colaborativo utilizando GitHub.

## Estructura del repositorio

- `src` → código del proyecto  
- `docs` → documentación del proyecto
